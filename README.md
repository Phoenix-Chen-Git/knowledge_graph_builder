# Copilot (Unlocked Agent Edition)

> [!NOTE]
> This is a modified fork of the [Obsidian Copilot](https://github.com/logancyang/obsidian-copilot) plugin by Logan Yang. All credit for the original core architecture goes to the original author.

**Copilot (Unlocked)** is an enhanced version of the Obsidian Copilot plugin that unlocks the powerful "Agent Mode" and removes "Plus" subscription restrictions for local power users.

- **üß† Local & Private**: Designed to work with your local models (via Ollama, etc.) or your own API keys without requiring a secondary subscription.

## üì¶ Installation

This plugin is not available in the official Obsidian directory (due to being a fork). You can install it using **BRAT**:

1.  Install **BRAT** from the Obsidian Community Plugins list.
2.  Open **Settings > BRAT**.
3.  Click **Add Beta plugin**.
4.  Paste the URL of this repository: `https://github.com/<your-username>/<your-repo-name>` (Update this after you publish!)
5.  Click **Add Plugin**.
